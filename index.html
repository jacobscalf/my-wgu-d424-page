JScalf D424 Project

Welcome to the JScalf D424 Project, an academic management app designed to help users track and manage terms, courses, and assessments. This app allows you to:

  • Log in securely with a username and password.
  • For testing purposes you can use "test" for the username and "password" for the password.
  • Manage academic terms, including adding and editing term details.
  • Add and edit courses within terms, with details like instructor information and course status.
  • Create and manage assessments for each course, with type restrictions.
  • Search for terms, courses, and assessments using a powerful search feature.
